## Documentation

### Container
Centralize content of your page using the `.container`.
```html	
<div class="container">
	<!-- contents here -->
</div>
```
### Grid
The grid is a basic system, creating columns `.column` and lines `.row`.
```html	
<div class="row">
	<div class="column">
		.column
	</div>
</div>
<div class="row">
	<div class="column column-3">
		.column .column-3
	</div>
	<div class="column column-3">
		.column .column-3
	</div>
	<div class="column column-3">
		.column .column-3
	</div>
	<div class="column column-3">
		.column .column-3
	</div>
</div>
<div class="row">
	<div class="column column-6">
		.column .column-6
	</div>
	<div class="column column-6">
		.column .column-6
	</div>
</div>
<div class="row">
	<div class="column column-8">
		.column .column-4
	</div>
	<div class="column column-4">
		.column .column-4
	</div>
</div>
```
